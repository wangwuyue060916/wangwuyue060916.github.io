wordpress搭建教程

发表于2023-02-08|更新于2023-02-18|wordpress
|阅读量:

安装宝塔控制面板

安装宝塔 Linux 面板，复制安装代码到 Xshell 回车运行。

中间需要进行 “确认”，输入 Y 回车即可。安装完成会显示登录信息，包括面板地址、用户名和密码。

打开浏览器访问该网址，登陆面板，会提示安装 LNMP。

将 PHP 版本改为 7.1，点击 “一键安装”，等待网站环境安装完成。

安装 WordPress

因为需要到 wordpress 官网下载程序，

解压完成后网站根目录会多出一个 wordpress 的文件夹

但这样不能直接使用的，我们还要继续将 wordpress 文件夹内的所有文件移动到网站的根目录

选中所有文件，然后剪切，然后到网站根目录粘贴所有

# WordPress 安装

环境和程序都就绪了，接下来就可以开始安装 Wordpress 网站了

访问 网站域名进入程序安装界面，第一步选择 现在就开始

WordPress 网站搭建成功，登陆 域名 /wp-login.php 管理网站
