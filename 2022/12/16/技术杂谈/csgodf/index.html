CSGO服务器一机多服的方法

发表于2022-12-16|更新于2023-01-06|技术杂谈CSGO
|阅读量:

# CSGO 的一机多服的方法有 2 种.

# 方法一:

多份 server.cfg 来实现不同” 密钥” 开多服的效果，
优点：节约磁盘空间，配置统一。所有服务器设置一次即可完成.
缺点：每个服没有办法独立个性化，
方法二：多份服务端，来实现不同” 密钥” 开多服的效果，
优点：可以设置每个服务端不同的插件配置，
缺点：每个服务端都需要单独设置.

具体使用哪种方法，完全根据个人的需求，
例如
方法一：比较适合小固态硬盘，只有 120g 那种且只是单纯扩展服务器的数量，并不需要额外修改.
方法二：比较适合需要对外出租的，每个客户对服务端需求不同，所以需要单独的服务端单独配置插件.

方法一的 CSGO 一机多服设置方法:
需要准备 2 个启动脚本，2 个 server.cfg 的文件。当如你要创建更多个服务器，就准备更多个脚本和文件.

# 第一步

路径:

D:\steamcmd\steamapps\common\Counter-Strike Global Offensive Beta – Dedicated Server
复制代码

创建

# 1 服启动脚本.bat

参数:

@echo off
cd /d %~dp0
cd “D:\steamcmd\steamapps\common\Counter-Strike Global Offensive Beta – Dedicated Server”
start srcds.exe -game csgo -console -ip 0.0.0.0 -usercon +game_type 0 +game_mode 0 +port 27015 +map de_dust2 -tickrate 128 +exec 1fserver.cfg -maxplayers_override 32 +mapgroup mg_active

复制代码

# 2 服启动脚本.bat

参数:

@echo off
cd /d %~dp0
cd “D:\steamcmd\steamapps\common\Counter-Strike Global Offensive Beta – Dedicated Server”
start srcds.exe -game csgo -console -ip 0.0.0.0 -usercon +game_type 0 +game_mode 0 +port 28015 +map de_dust2 -tickrate 128 +exec 2fserver.cfg -maxplayers_override 32 +mapgroup mg_active

复制代码

脚本里添加关键参数：一个是加载 cfg 文件参数和一个端口参数，每个服务器的端口必须不同.
1 服的

+exec 1fserver.cfg +port 27015

复制代码

2 服的

+exec 2fserver.cfg +port 28015

复制代码

# 第二步:

路径:

D:\steamcmd\steamapps\common\Counter-Strike Global Offensive Beta – Dedicated Server\csgo\cfg
复制代码

创建:
1fserver.cfg 文件

// CSGO 服务器登录令牌
sv_setsteamaccount “1277D8D73CD8A8D224746686F7E9C026”
// 系统参数
hostname “1 服 Xiongtianqi.cn” // 服务器名称将显示在服务器浏览器中。
复制代码

2fserver.cfg 文件

// CSGO 服务器登录令牌

sv_setsteamaccount “2277D8D73CD8A8D224746686F7E9C026”

// 系统参数

hostname “服务器名称”  //服务器名称将显示在服务器浏览器中。

复制代码

这样启动 2 个脚本基本就完成了，2 个服除了名字和端口不同，其它配置完全一致。没办法不同.

# 第三步:

启动 1 服和 2 服的开服脚本即可.

方法二的 CSGO 一机多服设置方法:
第一步：只需要将原本的服务端复制一份，将服务端目录名字区分开，例如 1csgoserver 和 2csgoserver.
第二步：将 1 服启动脚本和 2 服的启动脚本端口设置不同即可，例如：1 服的是 + port 27015 2 服的是 + port 28015
第三步：2 个服务器的 server.cfg 设置不同的密钥，
第四步：分别启动 2 个服务端的启动脚本，即可完成.
