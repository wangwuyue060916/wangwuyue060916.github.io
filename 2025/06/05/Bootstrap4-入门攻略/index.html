Bootstrap4 入门攻略

一、Bootstrap4 简介

Bootstrap 是全球最受欢迎的前端组件库，用于开发响应式布局和移动设备优先的 Web 项目。
Bootstrap4 是其第四代版本，基于 HTML、CSS 和 JavaScript 开发，具备以下核心特性：

  • Sass 变量与 Mixin：支持自定义样式，提升开发效率。
  • 响应式栅格系统：自动适配不同屏幕尺寸（如手机、平板、桌面）。
  • 预制组件：包含导航栏、按钮、表单、模态框等可复用组件。
  • jQuery 插件：提供轮播图、提示框、弹出框等交互功能（需单独引入 jQuery）。
  • 轻量化：相比 Bootstrap3，CSS 文件体积减少 40% 以上。
  • 浏览器支持：放弃对 IE8 和 iOS 6 的支持，最低支持 IE9+ 和 **iOS 7+**。

二、适合人群与前置知识

（一）适合人群

  • 具备基础 HTML/CSS 知识的前端开发者。
  • 希望快速搭建响应式网站的新手。
  • 需要开发移动优先项目的工程师。

（二）前置知识

  • 必须掌握：HTML 标签语法、CSS 盒模型与选择器、JavaScript 基础逻辑。
  • 推荐先学：
    • HTML 教程
    • CSS 教程
    • JavaScript 教程

四、核心知识点概览

（一）栅格系统

Bootstrap4 采用 12 列栅格布局，通过不同屏幕尺寸的类名实现响应式：

类名前缀 屏幕尺寸 应用场景
col- 所有尺寸 超小屏幕（手机）
col-sm- ≥576px 小屏幕（平板）
col-md- ≥768px 中等屏幕（桌面）
col-lg- ≥992px 大屏幕
col-xl- ≥1200px 超大屏幕

（二）常用组件

  1. 按钮（Button）

    • 颜色类：btn-primary（主色）、btn-secondary（次色）、btn-success（成功色）等。

  2. 导航栏（Navbar）

  3. 卡片（Card）
