Windows搭建我的世界服务器教程

发表于2022-11-05|更新于2023-01-05|技术杂谈
|阅读量:

# 准备

首先你得有台服务器（已经有的可以忽略），推荐阿里云、腾讯云、优刻得和华为云的

如果你是新用户有优惠，建议能买多长时间就买多长时间，因为优惠价的服务器到期后续费很贵的，而且新用户优惠只能买一次，所以尽量买长时间点划算。

2 核 4G 内存 5 兆带宽的服务器大约可以同时在线 5 到 10 人，4 核 8G 10 兆的服务器大约可以同时在线 10 到 30 人。1.18.2 版的配置要求更高了，最低得 4G 内存，建议 8G 内存。

# 购买服务器

选择安装 Windows Server 2019 或 2016 系统，然后用 RDP（远程桌面）登陆服务器。

如下图所示，搜索 RDP 然后打开，输入你刚买的服务器的 IP，然后输入你设的账号密码，如果不知道密码就到服务器控制台重置一下密码。账号一般默认是：administrator
安装 JAVA 环境
1.18.2 版需要 JDK17 版，需要安装其他版本 JDK 的看这篇文章：https://blog.zeruns.tech/archives/653.html

1.17 版需要 JDK16，1.16 版需要 JDK11，1.16 以下的版本需要 JDK8。

在服务器里打开 IE 浏览器，打开链接：https://url.zeruns.tech/Tuna_JDK

进入到下图所示目录里，然后下载红框里的.msi 后缀的 JDK17 安装包，点击另存为，自己选一个目录保存，或放到桌面。

然后双击打开刚刚下载的 JDK 安装包，一直点下一步就行。
端口开放
游戏服务器默认端口是 25565，需要开放 25565 端口，如果你是要用其他端口就开放对应端口。

如果你使用阿里云，腾讯云或者其他云服务商，请进入控制台到防火墙安全组策略，放行以上端口。

或者放行全部端口，入和出方向都要

然后还要关闭服务器系统自带防火墙或设置防火墙开放你要用的端口，win 系统默认关闭所有端口，除了 3389。

# 打开控制面板

点击 系统和安全。

然后接着点击 Windows Defender 防火墙 → 启用或关闭 Windows Defender 防火墙。
然后将这两个都选择关闭，点击确定即可。
安装 MC 官方服务端
安装官方服务端 和下面的 安装 forge 版服务端 选其中一个执行就行。

# 在服务器中下载 Minecraft1.18.2 服务端核心：

官网下载地址：https://www.minecraft.net/zh-hans/download/server
在服务器中找个地方新建一个文件夹做为 mc 服务端运行的目录，文件夹名字随便设，最好不要带中文。然后将上面下载的服务端核心文件 server.jar 复制到刚刚创建的文件夹里，在文件夹里新建一个文本文档 run.txt。

接着打开新建的 run.txt，在里面输入下面的启动命令，然后保存

java -server -jar server.jar nogui
pause

接着将 run.txt 文件重命名为 run.bat，然后双击运行 run.bat 即可启动服务端。
待运行完成后，打开 eula.txt 文件将 false 改成 true ，然后保存，接着再运行一次 run.bat，出现如下图所示信息即开服成功。若需要修改服务器端口和关闭正版验证之类的可以修改 server.properties 文件，然后重启服务端。

到这里就开服完成了，可以进入游戏了，在游戏中添加服务器，直接输入你服务器的 IP 即可，如果修改了端口就需要加上端口。

# 安装 forge 版服务端

安装 forge 版服务端 和上面的 安装官方服务端 选其中一个执行就行。

forge 版的支持安装 MOD。

在服务器中下载 forge 版 1.18.2 服务端安装器：

官网下载地址：https://files.minecraftforge.net/net/minecraftforge/forge/
安装和上述步骤一样，不再阐述
到这里就开服完成了，可以进入游戏了，在游戏中添加服务器，直接输入你服务器的 IP 即可，如果修改了端口就需要加上端口。
