（二）核心参数解析

参数名称 作用说明 示例值
id 国策 / 国策树唯一标识 id = "GER_rearmament"
country 归属国家与权重配置（factor需保持 0，modifier.add控制阶段性权重） tag = GER, add = 5
prerequisite 前置国策条件（OR用逗号分隔，AND用多个prerequisite标签） focus = "GER_industrial_expansion"
cost 国策持续时间（1=7天，支持小数） cost = 10（70 天）
ai_will_do AI 选择倾向（base基础值，modifier条件加成） base = 1, add = 0.5 if = { has_idea = "war_economy" }
completion_reward 完成国策奖励（支持事件触发、国家精神添加等） country_event = "GER_army_reform"

三、常用效果代码速查表

（一）国家与政治

功能 代码示例 说明
更改国名 set_cosmetic_tag = "NEW_TAG_DEF" 需同时修改本地化文件中的_DEF和_ADJ
改变意识形态 set_politics = { ruling_party = communist } 切换执政党（如共产主义）
增加支持度 add_popularity = { ideology = democratic popularity = 0.2 } 民主党支持度 + 20%
发动内战 start_civil_war = { ideology = fascist size = 0.8 } 法西斯势力发动内战，初始势力 80%

（二）外交与战争

功能 代码示例 说明
签订互不侵犯条约 diplomatic_relation = { country = FRA relation = non_aggression_pact active = yes } 与法国签订条约
创建战争目标 create_wargoal = { type = take_state_focus target = BEL expire = 14 } 获得比利时征服目标（14 天有效期）
释放傀儡国 宗主国 = { set_autonomy = { target = CZE autonomy_state = autonomy_free } } 释放捷克并赋予完全自治
增加世界紧张度 add_named_threat = { threat = 0.5 name = "重整军备" } 紧张度 + 0.5，原因标注 “重整军备”

（三）经济与资源

功能 代码示例 说明
增加人力 add_manpower = 20000 人力 + 20000
修建铁路 build_railway = { level = 3 path = { 101 102 103 } } 在地块 101-103 间修建 3 级铁路
资源使用权转让 give_resource_rights = { receiver = ITA state = 48 } 将地块 48 的资源使用权交给意大利
增加建筑位 add_extra_state_shared_building_slots = 5 in = 77 地块 77 增加 5 个建筑位

（四）科技与军事

功能 代码示例 说明
增加科技加成卡 add_tech_bonus = { category = "armor" bonus = 0.1 uses = 2 } 装甲科技加成 + 10%，2 张加成卡
生产装备储备 add_equipment_to_stockpile = { type = medium_tank_chassis amount = 1000 producer = GER } 德国生产 1000 辆中坦底盘
解锁派遣志愿军 set_rule = { can_send_volunteers = yes } 允许派遣志愿军

四、进阶技巧与注意事项

（一）定位系统详解

  1. 绝对定位：以国策树左上角为原点（x=0, y=0），y值越大越靠下，x值越大越靠右。

  2. 相对定位：基于指定国策的坐标偏移（如relative_position_id = "parent_focus"），适用于分支国策布局。

（三）调试与排错

  1. 实时刷新：修改国策后需手动刷新国策树（快捷键F11或重新加载游戏）。
  2. 冲突排查：
    • 互斥国策需双向配置mutually_exclusive。
    • 动态图标trigger条件需确保至少一个默认生效（如always = yes）。
  3. 日志查看：游戏根目录logs/focus.txt记录国策加载错误信息。
